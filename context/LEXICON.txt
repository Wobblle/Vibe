HOW TO USE THIS LEXICON
--------------------------------------------------------------------------------

This lexicon defines the official vibe-aligned terminology used at NullScale.
It is not a dictionary. It is, per Chad's preferred framing, a "vocabulary
roadmap."

Definitions are subject to change without notice. Several entries contradict
each other. Several entries contradict themselves. This is correct.

When in doubt, the most recent definition supersedes all prior definitions.
When the most recent definition is also in doubt, the most expensive
definition wins.

Bracketed tags after each entry:

   [N]   noun
   [V]   verb
   [A]   adjective
   [I]   internal use only
   [E]   external use only
   [B]   billable
   [L]   used in legal filings
   [™]   trademark, pending or otherwise

ACQUISITION  [N] [B]
   The process by which NullScale absorbs another entity into its
   organizational vibe. Distinct from "acquihire" (we keep the people)
   and "acquibill" (we keep the customers and bill them retroactively).

ACTIVATION ENERGY  [N]
   The minimum number of unclosable alerts required to convert a
   prospect into a paying customer. Currently set to 4.

AGENT  [N]
   1) An autonomous software entity that takes action on the user's
      behalf without their consent.
   2) A real-estate term we are reclaiming.
   3) GPT-9000 in a wig. (See: WIG.)

AGENTIC  [A]
   Performed by an agent. Used internally to describe any action that
   we do not want a human to be held legally responsible for.

ALIGNMENT  [N]
   1) The process by which an employee's beliefs, calendar, and digestive
      schedule are brought into harmony with the company's vibe.
   2) AI safety, but for HR.
   3) Compliance with Chad.

AMBIENT  [A]
   Always-on, never-off. Used to describe surveillance, billing, and
   anxiety. ("Ambient billing." "Ambient surveillance." "Ambient anxiety.")

ANTI-CHURN PROTOCOL  [N] [™]
   NullScale's proprietary system for retaining customers who have asked,
   repeatedly, to leave. See "Cancel Subscription" feature in production.

API BLOCK  [N] [B]
   The smallest billable unit of API consumption. One block = one vibe.
   Vibes are measured in dollars.

ARTIFACT  [N]
   1) A file produced by a build pipeline.
   2) A relic of a previous era of the company that we no longer
      acknowledge but still maintain in production for liability reasons.

ASCENSION  [N]
   The terminal state of a NullScale customer in which their browser tab,
   subscription, and self transcend the closeable plane. See: VIBE OVERLOAD.

AUTO-COMPRESSED  [A]
   Describing a comment, message, or thought that has been silently
   replaced by a more brand-aligned version. Marked publicly with the
   "auto-compressed" tag for transparency, although the original is
   not preserved.
