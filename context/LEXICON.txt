HOW TO USE THIS LEXICON
--------------------------------------------------------------------------------

This lexicon defines the official vibe-aligned terminology used at Vibratur.
It is not a dictionary. It is, per Chad's preferred framing, a "vocabulary
roadmap."

Definitions are subject to change without notice. Several entries contradict
each other. Several entries contradict themselves. This is correct.

When in doubt, the most recent definition supersedes all prior definitions.
When the most recent definition is also in doubt, the most expensive
definition wins.

Bracketed tags after each entry:

   [N]   noun
   [V]   verb
   [A]   adjective
   [I]   internal use only
   [E]   external use only
   [B]   billable
   [L]   used in legal filings
   [™]   trademark, pending or otherwise
